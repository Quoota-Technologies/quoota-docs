# Rituales de equipo

En esta sección se describen los rituales o ceremonias que forman parte del proceso de trabajo del equipo. Estos rituales están diseñados para definir la metodología de trabajo que lleva el equipo en la gestión de proyectos y tareas. 

Cada documento detalla un ritual específico, incluyendo su propósito, roles participantes, frecuencia y estructura. A continuación, se presentan los rituales principales:

- [Daily Standup](daily-standup.md): Reunión diaria breve para sincronizar al equipo, identificar obstáculos y planificar las tareas del día.
- [Sprint Planning](sprint-planning.md): Reunión al inicio de cada sprint para definir el trabajo a realizar durante el sprint.
- [Sprint Review](sprint-review.md): Reunión al final de cada sprint para revisar el trabajo completado, obtener feedback y planificar
- [Sprint Retrospective](sprint-retro.md): Reunión al final de cada sprint para reflexionar sobre el proceso de trabajo del equipo, identificar áreas de mejora y planificar acciones para el próximo sprint.
